# ShieldCore LA — UniFi Installation Website

## Files
- `index.html` — Main website
- `planner.html` — Camera Placement Planner (Demo)
- `slide1.png / slide2.png / slide3.png` — Hero slideshow screenshots
- `*.png` — 76 product images (placeholders, replace with real via fetch_images.py)
- `fetch_images.py` — Run once to download real images, then delete

## Setup
1. Run `python3 fetch_images.py` to get real product images
2. Delete `fetch_images.py`
3. Push to GitHub → connect to Cloudflare Pages (no build settings) → live
